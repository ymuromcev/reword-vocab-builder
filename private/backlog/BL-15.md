## Context

Финальный шаг — выложить репо на GitHub публично. Делаем **только
после** того, как CLI работает end-to-end и README имеет
квалифицированные примеры (BL-14).

## Образ результата

```bash
gh repo create ymuromcev/reword-vocab-builder --public --source=. --remote=origin --push
```

Репо доступен на https://github.com/ymuromcev/reword-vocab-builder с
README, лицензией и initial commit.
