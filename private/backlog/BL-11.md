## Context

Собрать все компоненты в один CLI. Без этого ничего не работает.

## Образ результата

```bash
reword-vocab topic "PM interview vocabulary"
# → output/2026-05-19-pm-interview-vocabulary.csv
# → output/2026-05-19-pm-interview-vocabulary-flagged.txt (если есть)
# → 152 слова сгенерировано, 47 отброшено как известные (dedup)
# → CSV готов к импорту в Reword

reword-vocab source ~/Documents/inspired.pdf --instruction "PM vocabulary"
# → output/2026-05-19-inspired-pm-vocabulary.csv
# → ...
```

## Plan

- [x] `src/cli.py`: argparse. Subcommands `topic`, `source`.
- [x] Pipeline (общий):
      1. `fetch_latest_backup()` (BL-03) → `read_backup()` (BL-02).
      2. Генерация word_list через `topic.generate()` или
         `source.extract()`.
      3. Dedup через `dedup()` (BL-06).
      4. Enrich: IPA (BL-09) + examples/translation (BL-10).
      5. Verb prefix (BL-04).
      6. Write CSV (BL-05).
- [x] Опции: `--output PATH`, `--target-count N`, `--backup-path PATH`.
      `--no-dedup` отказались (нет use-case).
- [x] Progress lines (вместо tqdm — без дополнительной зависимости).
- [x] Итоговый отчёт: kept / skipped / flagged-for-review counts.
