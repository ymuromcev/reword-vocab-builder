## Context

Скилл — тонкий слой, который позволяет вызывать тул из Claude Code
естественным языком ("build me a vocab for X"), не выходя в шелл.

## Образ результата

В Claude Code:

```
> build me a vocabulary for system design interview
```

Claude вызывает `reword-vocab topic "system design interview"`,
показывает прогресс, в конце даёт ссылку на сгенерированный CSV.
