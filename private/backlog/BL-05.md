## Context

Reword принимает на импорт только специфический формат CSV. Любое
отклонение (header, single-quote, comma-separated) → парсер падает или
теряет данные.

## Образ результата

```python
rows = [
    {"word": "to leverage", "ipa": "[ˈlevərɪdʒ]",
     "ru": "использовать", "ex1_en": "...", "ex1_ru": "...",
     "ex2_en": "...", "ex2_ru": "..."},
    ...
]
write_csv(rows, "output/pm-vocab-2026-05-19.csv")
```

Результат: файл точно в формате Reword, проверяется на эталоне
`~/Desktop/pm_interview_vocab.csv`.
