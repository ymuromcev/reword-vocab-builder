## Context

Reword хранит прогресс в SQLite-бэкапе (`reword_en.backup`). Внутри —
таблица `WORD` с полями `WORD, S_REC, S_REP, I_REC, I_REP, E_REC,
E_REP` (последние два — interval в секундах). Нам нужно прочитать
бэкап и для каждого слова определить состояние выученности.

## Образ результата

Чистая Python-функция:

```python
words = read_backup("path/to/reword_en.backup")
# → dict: word_lowercase → ClassifiedWord(word, status, interval_days)
```

Status — один из: `mastered`, `active-long`, `active`,
`passive-mastered`, `passive-long`, `passive`, `seen-only`.

Тесты используют fake SQLite-файл с известными значениями интервалов
и проверяют пороги.

## Notes

Опорная логика порогов:

| status | условие |
|---|---|
| mastered | studied (S_REP > 0) AND I_REP >= 60 дней |
| active-long | studied AND I_REP >= 14 дней |
| active | studied AND I_REP < 14 дней |
| passive-mastered | recognized (S_REC > 0, S_REP = 0) AND I_REC >= 60 дней |
| passive-long | recognized AND I_REC >= 14 дней |
| passive | recognized AND I_REC < 14 дней |
| seen-only | в словаре, но не изучалось |
