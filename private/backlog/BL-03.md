## Context

Бэкап Reword лежит на Google Drive юзера (или в iCloud). Тул должен
сам найти последний и скачать его локально, чтобы дальше его прочитал
BL-02. Доступ — через Google Drive MCP-коннектор, без явных токенов.

## Образ результата

Функция:

```python
local_path = fetch_latest_backup()
# → '/tmp/reword_en.backup' (свежий)
```

CLI при старте вызывает её, кеширует на N часов (чтобы не качать
каждый раз), и передаёт путь в BL-02 reader.

## Plan

- [ ] `src/drive_mcp.py`: поиск файла `reword_en.backup` на Drive
      (по имени или конфигурируемой папке).
- [ ] Скачивание во временный файл / кеш-директорию
      `~/.cache/reword-vocab-builder/`.

- [ ] Конфиг через ENV: `REWORD_BACKUP_PATH` для override (если юзер
