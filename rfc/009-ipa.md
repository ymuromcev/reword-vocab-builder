## Goal

Return a US-English IPA transcription wrapped in square brackets, plus a
`flagged_for_review` boolean for words that fell back to LLM.

## Public API

```python
from src.ipa import transcribe

ipa, flagged = transcribe("leverage")     # ("[ˈlevərɪdʒ]", False)
ipa, flagged = transcribe("hyperscaler")  # ("[ˌhaɪpərˈskeɪlər]", True)
ipa, flagged = transcribe("xyzqq")        # (None, True)
```

When `ipa is None`, the word is dropped from the CSV. When `flagged` is
True and `ipa` is not None, the CSV row is written with the prefix
`[? to verify] ` so the user sees it on import.

## Pipeline

1. **Normalize** — strip leading `to `, lowercase, strip whitespace.
2. **Single word** — look up in CMU dict (`cmudict` PyPI package,
   offline). Convert ARPAbet → IPA via a fixed table baked into the
   module. Use the first pronunciation if multiple are present.
3. **Phrasal (multi-word)** — transcribe each word independently, join
   with single space inside the brackets.
4. **LLM fallback** — single prompt:
   "Return US-English IPA in square brackets, no other text: `{word}`".
   Set `flagged_for_review=True`.
5. **Sanity check on LLM output** — must match `^\[[^\[\]]+\]$`. If
   not, return `(None, True)`.

## ARPAbet → IPA table

Static dict at module top. Stress markers preserved (`AH1` → `ˈʌ`,
`AH0` → `ə`). Document the source in module docstring (CMU dict README
references the mapping).

## Flagged manifest

`transcribe` is pure. The caller (CLI) collects flagged words and writes
them to `output/<run-id>-flagged.txt`. Not the IPA module's concern.

## Tests

- 100-word fixture covering common PM vocabulary; assert ≥90% resolve
  from CMU without LLM fallback.
- Phrasal verbs: `circle back`, `dive in`.
- Idioms: `move the needle` → space-joined transcription.
- Stress marker conversion: `leverage` → `[ˈlevərɪdʒ]` not `[levərɪdʒ]`.
- LLM fallback: mocked client, verify `flagged=True` and bracket-only
  output.
- Garbage input (`xyzqq`) — LLM returns junk → `(None, True)`.

## Out of scope

- UK accent (P3).
- Homograph disambiguation (`read` past vs. present).

## Risks / decisions

- **CMU first** — offline, deterministic, no token cost on common
  words. Decisive quality win.
- **Square brackets vs slashes** — Reword expects brackets, match that.
- **Drop word if both CMU and LLM fail** — better than ship an empty
  IPA field.
